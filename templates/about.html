About This Site

This site monitors the City of {{ city.name }} Agenda Center for new city council meeting agendas and minutes.

How It Works

  1. Scraping: The site periodically checks the city's website for new agenda and minutes PDFs.
  2. Summarization: New documents are sent to Claude (an AI assistant by Anthropic) which reads the PDF and generates a plain-language summary.
  3. Notification: Subscribers are notified via email and/or Bluesky when new summaries are available.

Important Notes

  • Summaries are AI-generated and may contain errors or omissions. Always refer to the original PDF documents for official information.
  • This site is not affiliated with the City of {{ city.name }}. It is an independent project.

Source Code

This project is open source. The code is available on GitHub.
